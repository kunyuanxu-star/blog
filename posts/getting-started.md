# 开始使用

欢迎使用这个基于 VitePress 的博客！

## Markdown 语法支持

这个博客完全支持 Markdown 语法，你可以轻松地撰写博客文章。

### 标题

使用 `#` 符号创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 代码块

支持代码高亮：

```javascript
function hello() {
  console.log('Hello, World!');
}
```

```python
def hello():
    print("Hello, World!")
```

### 列表

无序列表：
- 项目 1
- 项目 2
- 项目 3

有序列表：
1. 第一项
2. 第二项
3. 第三项

## 如何撰写新文章

1. 在 `posts/` 目录下创建新的 `.md` 文件
2. 使用 Markdown 语法撰写内容
3. 在 `.vitepress/config.mts` 中更新侧边栏配置
4. 提交到 GitHub，自动部署

就是这么简单！
